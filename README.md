This is my first project as a developer. In this project, I created a game called “Secret Number Game,” where the user interacts with the interface in an attempt to guess a hidden number between 1 and 100.

The game was designed to be simple but engaging. The player inputs guesses, and the system provides feedback indicating whether the chosen number is higher or lower than the secret number. This interactive loop continues until the correct number is guessed, encouraging logical thinking and incremental problem-solving.

To build this project, I applied foundational knowledge of JavaScript, HTML, and CSS. JavaScript was used to handle the game logic, including generating the random number and validating user input. HTML provided the structure of the interface, while CSS was used to style the layout and improve the overall user experience.

Although it is a beginner project, it represents an important step in my journey as a developer, as it combines core programming concepts such as conditionals, user interaction, and basic front-end development into a complete, functional application.
